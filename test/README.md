## Run Instructions
After building the project, the executable `main` is automatically copied into this directory.

### Step 1: Prepare Input NC File
Download the large NC input file named **data_extraction_cable_model_calibration_cluster_new.nc** from Google Drive:
[Google Drive Download Link](https://drive.google.com/file/d/1RrnM_XRPshyo8hhxZFwIvXbjnEIGjSmI/view?usp=drive_link)

Put the downloaded NC file into the `/test/input` directory.

### Step 2: Execute the Program
Once the input file is properly placed, run the startup script directly via the command below:
```bash
./rum_main.sh
```
